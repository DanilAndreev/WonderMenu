# WonderMenu C++
### Wonder Menu - a console menu builder for c++

### Features:
* Supports keyword for process end
* Supports unlimited quantity of user maked command, just inherit your class from __MenuCommand__ and override __handleMessage(...)__ method
* Supports user styled print, just inherit your class from __WonderMenu__ and override __print()__ method
